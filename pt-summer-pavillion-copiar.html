Summer Pavillion

O projeto foi realizado para um concurso do Arquideas, em 2016. O objetivo do concurso passava por criar, numa escala moderada, um espaço versátil, no emblemático Central Park, que pudesse acolher diferentes atividades no verão.

Este projeto quer ter o menor impacto visual na paisagem e integrar-se no ambiente natural do parque. Desta forma, espelhos a todo o perímetro permitem que o pavilhão se misture por entre as árvores. Os interiores brancos e a "torre" são os únicos elementos que revelam que há algo escondido naquele local. O pavilhão abriga uma grande rocha local, dando sentido ao projeto e servindo como âncora ao local escolhido para o pavilhão.

 

 

Neste espaço, é possível realizar diversas atividades; é proposta uma sala polivalente, uma sala de exposições, um pequeno teatro e um restaurante com capacidade para pequenos concertos. O pavilhão é flexível o suficiente para se adaptar a uma grande variedade de atividades.

A sustentabilidade é um fator chave no projeto. Além de não precisar de uma preparação invasiva do terreno, foi escolhida a madeira como material de construção por esta poder ser reciclada. Como extra, os espelhos de fachada ajudam a refletir o calor, economizando energia no verão.

Este espaço pretende ser um ponto de referência cultural e de lazer para os habitantes e visitantes de Nova Iorque.
